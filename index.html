HOW TO USE — SCHWAB API SETUP GUIDE
Schwab access tokens expire every 30 minutes. Refresh tokens last 7 days. You will need to refresh your token roughly once per week. The scanner will show an error in the log when your token expires.
1

Create a Schwab developer account

Go to developer.schwab.com and sign in with your regular Schwab brokerage account. Accept the developer agreement.

2

Create a new app

Click "My Apps" → "Create App." Name it anything (e.g. PetroAI Scanner). Set the callback URL to exactly: https://127.0.0.1 — must match exactly.

3

Copy your App Key and Secret

After creating the app you get your App Key (Client ID) and App Secret. Paste both into Settings above. Treat these like passwords.

4

Authorize and get your access token

In a new browser tab, paste this URL replacing YOUR_KEY with your App Key:
https://api.schwabapi.com/v1/oauth/authorize?client_id=YOUR_KEY&redirect_uri=https://127.0.0.1&response_type=code&scope=readonly
Log in when prompted. You'll land on a broken page — that's normal. Copy the full URL from the address bar. It contains code=XXXXXX. That code is temporary — use it within 30 seconds.

5

Exchange the code for a token (use Postman)

Download Postman (free at postman.com). Create a POST request to: https://api.schwabapi.com/v1/oauth/token with your App Key, Secret, and the code. You'll receive an access_token. Paste that into Settings → Access Token field.

6

Save and test

In Settings, click "Save & Apply" then "Test Connection." You should see a green LED and "API: Connected" in the top bar. You're live.

7

Run your first scan

Click RUN SCAN. The scanner loops through all PetroAI tickers, fetches the options chain for Friday's expiration, and flags any strikes with unusual volume-to-open-interest ratios. Green rows = hot alerts. Amber = watch.

8

Reading the alerts

Green border = Vol/OI above 10x — highest priority. SWEEP = one large aggressive order. ACCUM = volume building over multiple prints. Always verify the setup on your chart before entering. Max $50–$100 premium per contract.
